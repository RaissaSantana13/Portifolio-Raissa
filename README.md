## Funcionalidades e Secções

O portfólio foi desenhado com foco em usabilidade e estética moderna, apresentando:

* **Apresentação Profissional:** Secção "Sobre Mim" detalhando a minha formação e objetivos na área tecnológica.
* **Showcase de Projetos:** Galeria interativa com cartões detalhados para os principais projetos:
    * **Termo MVP:** Jogo de palavras em C#.
    * **Bookly:** Gestão de leituras e base de dados.
    * **Quiz App:** App educacional em Flutter com sistema de sorteio por dificuldade (5 fáceis, 3 médias e 2 difíceis) e embaralhamento de respostas para evitar a memorização.
    * **BiblioXchange:** Plataforma de troca de livros.
    * **Alter Ego:** Experiência interativa baseada na artista Lisa.
* **Habilidades Técnicas:** Listagem visual das linguagens e ferramentas que domino, como C#, SQL, HTML, CSS e JavaScript.
* **Navegação Fluida:** Header responsivo com efeito de mudança de cor ao fazer scroll e navegação suave (smooth scroll) entre secções.
* **Contacto Direto:** Links integrados para e-mail, GitHub e descarregamento de CV.

---

## Tecnologias Utilizadas

* **HTML5:** Estrutura semântica focada em acessibilidade.
* **CSS3 Avançado:**
    * **Layout:** Utilização de Flexbox e Grid para responsividade.
    * **Design:** Tema escuro com acentos em tons de roxo e azul, seguindo uma estética tecnológica.
    * **Efeitos:** Transições suaves em botões e cartões de projeto.
* **JavaScript:** Lógica para interatividade da interface e gestão de eventos de navegação.

---

## Estrutura de Diretórios

* `index.html`: Arquivo principal contendo toda a estrutura da página única (Single Page Application style).
* `style.css`: Estilização completa e regras de responsividade para dispositivos móveis.
* `script.js`: Implementação de efeitos dinâmicos e navegação.
* `assets/`: Repositório de imagens, ícones SVG e o ficheiro do currículo.

---

## Como Visualizar

Para abrir o portfólio localmente:
1. Faça o clone do repositório.
2. Abra o ficheiro `index.html` em qualquer navegador moderno.
3. Ou aceda diretamente via **GitHub Pages** (se configurado).
